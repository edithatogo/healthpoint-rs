# Contracts

This directory contains interim, checked-in integration contracts that can be used before the Rust CLI has generated canonical schemas.

- `schemas/*.schema.json` are generated by `scripts/generate-contract-schemas.py`.
- After Rust compilation is available, compare these with `healthpoint schema <target>` and replace any drift with CLI-generated output.
- These schemas describe the tool's typed projection, not the complete Healthpoint or FHIR specification.
